# Product Spec — 红薯创作坊 (XHS ReCreator)

## 一句话描述

基于多模态 AI 的小红书内容二创工具 — 粘贴链接，一键生成全新文案和配图。

## 目标用户

自己 / 小团队内部使用，私有部署。

## 核心价值

1. **降低创作门槛** — 不需要设计能力，AI 自动生成配图
2. **批量生产内容** — 从一条笔记快速生成一条风格全新的笔记
3. **风格可选** — 学霸笔记风 / 白板纪实风，后续可扩展

## 用户流程

```
粘贴小红书链接 → AI 抓取原文+原图 → 预览确认 → 选择风格 → AI 二创 → 查看结果
```

## 功能清单

### 已完成

| 功能 | 说明 |
|------|------|
| 笔记抓取 | 粘贴链接，自动抓取标题/文案/标签/图片 |
| 智能分析 | Vision AI 分析原图内容，生成详细描述 |
| 文案改写 | LLM 根据原文+图片分析生成新文案 |
| 图片生成 | AI 根据新文案生成全新配图 |
| 多风格支持 | 学霸笔记风 / 白板纪实风 |
| 历史记录 | SQLite 存储所有历史任务 |
| 实时进度 | WebSocket 推送任务进度 |
| 落地页 | 产品介绍首页 |
| 暗色模式 | 日/夜间模式切换 |

### 进行中（Login MVP）

| 功能 | 说明 |
|------|------|
| 用户注册/登录 | 用户名+密码，JWT 鉴权 |
| 任务隔离 | 每人只看自己的历史记录 |
| Cookie 隔离 | 每人配置自己的小红书 Cookie |

### 规划中（Post-MVP）

| 功能 | 说明 |
|------|------|
| 管理后台 | 管理员管理用户 |
| 注册开关 | 控制是否开放注册 |
| 更多图片风格 | 扩展风格库 |
| 批量二创 | 一次提交多个链接 |
